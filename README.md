# iGamePanel
iGamePanel est un panel assemblant WHMCS & Pterodactyl en un seul et unique panel et tous sa gratuitement ! Il est développer sur la base de Pterodactyl
Il est actuellement en développement.

### Jeux/Application disponible

* Minecraft — including Paper, Sponge, Bungeecord, Waterfall, and more
* Rust
* Terraria
* Teamspeak
* Mumble
* Team Fortress 2
* Counter Strike: Global Offensive
* Garry's Mod
* ARK: Survival Evolved
* Factorio
* San Andreas: MP
* Pocketmine MP
* Squad
* FiveM
* Xonotic
* Starmade
* Discord ATLBot, and most other Node.js/Python discord bots
